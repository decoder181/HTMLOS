What is HTMLOS for?

HTMLOS is meant to be used as a mini operating system. However, it is also meant to be used for fun projects, like learning to code in html, by adding new apps, adding new functionality to apps, and changing the design to be fit to one's interests. Please keep in mind that if you change the code of HTMLOS, please keep the name in order to credit the original creator.
